7 - Using React and widget-components

Widget-components is a collection of reusable components written with React framework. Currently the list involves following components: HeaderComponent and OutcomeComponent

Pre-requisites

You can start using widget-components by installing it as a dependency npm install widget-components react react-dom --save

Using components in a widget (ex. OutcomeComponent)

Open up the file where you would like to use the chosen component and add import { ComponentName } from 'widget-components' on top of the file. In order to properly use any of the components you need to follow instruction on the list of arguments that you need to supply.

For example OutcomeComponent have a three parameters that supply.

  • outcome (required)
  • event
  • withLabel
  • customLabel
You now have the possibility to use this component with the following syntax render() { return <OutcomeComponent argumentName={ argumentValue }></OutcomeComponent> } when use inside other component or as a argument to ReactDOM.render(<OutcomeComponent argumentName={ argumentValue }></OutcomeComponent>, htmlTag)
